Smart Traffic Control System 🚦

Overview

This project is developed using Python, OpenCV, and YOLOv8.

It detects vehicles from a traffic video and calculates the traffic signal timing based on the number of detected vehicles.

Features

- Vehicle Detection
- Vehicle Counting
- Dynamic Traffic Signal Timing
- Real-time Video Processing

Technologies Used

- Python
- OpenCV
- YOLOv8

Project Structure

- main.py
- detector.py
- traffic_logic.py
- yolov8n.pt

How to Run

1. Install the required libraries.
2. Place the traffic video inside the "videos" folder.
3. Run:

python main.py
