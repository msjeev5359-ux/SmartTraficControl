# Smart Traffic Control System 🚦

## Overview
This project detects vehicles from a traffic video using Python, OpenCV, and YOLOv8.

## Features
- Vehicle Detection
- Car Count
- Bus Detection
- Truck Detection
- Traffic Video Processing

## Technologies Used
- Python
- OpenCV
- YOLOv8

## Project Structure

SmartTrafficControl/
├── main.py
├── detector.py
├── traffic_logic.py
├── yolov8n.pt
├── videos/
└── README.md

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## Output
The program detects vehicles in the traffic video and displays the result in real time.
